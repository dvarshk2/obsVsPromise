Observable Vs Promises

Observable Vs Promises Difference

Promises

  1. Promises are eager
  2. Promises emit data one time only
  3. can not manipulate data using RxJs Operator
  4. Promises cannot be unsubscribed
  5. can not handle complex operations as compared to Observables

Observables

  1. Observables are lazy
  2. Observables emit data multiple times i.e. as data changes
  3. Can manipulate data using Rxjs Operator
  4. Observables can be unsubscribed
  5. Observables can handle complex operation as it has multiple Rxjs Operators
